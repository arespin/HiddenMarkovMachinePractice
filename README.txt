Contains:

TomSawyer.dx1 - to run compression.py on as the only argument


compression.py Ð Run this. The first argument should be the text file to run on (TomSawyer.dx1). The terminal will print the probability, c[i] and I[i] of each letter. 

out.txt Ð this is where compression.py prints to. It will have the word, the left to right lower and upper of the interval, and the right to left lower and upper of the interval. 

graphic.pdf - This graphs the left to right and right to left initial points for each word.


A program that analyzes letters as produced by two states, S0 and S1 in an state-emision hidden
Markov model. Each of the 2 states will produce each of the symbols in the alphabet with a non-zero
probability.
